# 终端贪吃蛇游戏规范

## 1. 项目概述

- **项目名称**: Terminal Snake
- **类型**: 终端命令行游戏
- **核心功能**: 经典贪吃蛇游戏,在终端中运行
- **目标用户**: 休闲玩家,学习Python的开发者

## 2. 技术规范

- **语言**: Python 3
- **依赖**: 仅使用标准库 (curses)
- **跨平台**: 支持 Linux/macOS (Windows 需要 windows-curses)

## 3. 功能规范

### 3.1 游戏机制
- 蛇初始长度为3节,出现在游戏区域中央
- 食物随机生成,不能与蛇体重叠
- 蛇移动速度: 初始150ms/格,每吃5个食物速度+10%
- 撞墙或撞自己则游戏结束
- 分数 = 吃到的食物数量 × 10

### 3.2 控制
- `↑↓←→` 或 `WASD`: 方向控制
- `空格`: 暂停/继续
- `Q` 或 `Esc`: 退出游戏
- `R`: 重新开始

### 3.3 显示
- 游戏区域: 40列 × 20行
- 蛇头: `@` (亮绿色)
- 蛇身: `o` (绿色)
- 食物: `*` (红色)
- 墙壁: `#` (灰色)
- 背景: `.` (浅色)

### 3.4 界面
- 显示当前分数
- 显示当前速度(刷新率)
- 游戏开始提示
- 游戏结束显示最终分数和重新开始提示

## 4. 验收标准

- [x] 蛇能正确移动并响应方向键
- [x] 食物随机生成且不与蛇体重叠
- [x] 吃到食物后蛇身增长,分数增加
- [x] 撞墙或撞自己时游戏结束
- [x] 暂停功能正常工作
- [x] 可以重新开始游戏
- [x] 速度随进度增加
