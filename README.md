# Terminal Snake Game

用 Python 编写的经典贪吃蛇游戏，支持终端运行。

## 功能特性

- **难度选择** - 简单/中等/困难三种模式
- **高分系统** - 本地持久化存储最高分
- **道具系统** - 随机生成道具：
  - `S` (紫色) - 减速道具
  - `D` (紫色) - 双倍分数
- **音效反馈** - 吃到食物/暂停/游戏结束提示音
- **输入缓冲** - 更流畅的控制体验
- **暂停功能** - 空格键暂停
- **游戏状态** - 菜单/游戏中/暂停/结束

## 操作方式

- `↑↓←→` 或 `WASD` - 方向控制
- `空格` - 暂停/继续
- `Q` - 返回菜单
- `R` - 重新开始
- `1/2/3` - 选择难度

## 环境要求

- Python 3.6+
- Linux/macOS：无需额外依赖
- Windows：需要安装 `windows-curses`

## 安装

```bash
# 克隆仓库
git clone https://github.com/morninghut/claude-snake-game.git
cd claude-snake-game
```

### Windows 额外安装

```bash
pip install windows-curses
```

## 运行

```bash
python snake_interactive.py
```

## 游戏界面

```
========================================
|                                    |
|           @  ooo  *                |
|                                    |
|    Score: 30 | High: 50 | Speed: 10|
|                                    |
========================================
```

### 符号说明

| 符号 | 含义 |
|------|------|
| `@` | 蛇头（绿色）|
| `o` | 蛇身（青色）|
| `*` | 食物（红色）|
| `S` | 减速道具（紫色）|
| `D` | 双倍分数道具（紫色）|
| `#` | 墙壁（白色）|
